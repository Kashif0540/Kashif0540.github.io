## ⚡ Features

- **Terminal-style hero** — animated typing effect that cycles through roles (Full Stack Developer, AI/LLM Engineer, MERN & Laravel Builder, Automation Architect)
- **Scroll-triggered reveals** — sections animate in on scroll using IntersectionObserver
- **Count-up stats** — numbers animate from 0 to their value when scrolled into view
- **Custom dark theme** — olive and bone color palette with a signal red accent
- **Scrolling tech marquee** — continuous horizontal ticker with inline SVG tech icons
- **Curtain preloader** — multi-column drop animation on page load
- **Fully responsive** — works down to 320px viewport width
- **Lightweight** — single HTML file, zero external JS libraries
